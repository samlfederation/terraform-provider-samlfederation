# OpenTofu Provider reading SAML federation metadata

> [!WARNING]
> This provider is still in development and not ready for use yet.

This OpenTofu provider reads SAML2 federation metadata. You can use it to configure [Keycloak](https://search.opentofu.org/provider/keycloak/keycloak/latest) or [Authentik](https://search.opentofu.org/provider/goauthentik/authentik/latest).
